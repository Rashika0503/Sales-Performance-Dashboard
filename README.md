# 📊 Sales Performance Dashboard

## 📌 Project Overview

This project presents an interactive **Sales Performance Dashboard** built in **Power BI** to analyze sales performance across different regions, categories, and time periods. The dashboard provides key business insights through KPI cards, interactive visualizations, and slicers, helping users understand sales trends and make data-driven decisions.

---

## 🎯 Objective

- Analyze overall sales performance.
- Monitor monthly sales trends.
- Compare sales across different regions.
- Analyze category-wise sales contribution.
- Provide interactive filtering using slicers.
- Generate business insights through visual analytics.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel / CSV Dataset

---

## 📂 Dataset

The dashboard is built using a Superstore Sales dataset containing the following fields:

- Order Date
- Region
- Category
- Product Name
- Sales
- Profit
- Quantity
- Customer Name
- Segment
- State
- City
- Ship Mode

---

## 📈 Dashboard Features

### KPI Cards
- Total Sales
- Total Orders
- Total Profit
- Total Quantity

### Visualizations
- 📉 Monthly Sales Trend (Line Chart)
- 📊 Sales by Region (Bar Chart)
- 🍩 Sales by Category (Donut Chart)

### Interactive Filters
- Region Slicer
- Category Slicer

---

## 📊 Key Business Insights

- East region emerged as the top-performing region in terms of total sales. 

- Office Supplies accounted for the largest share of overall sales. 

- Sales remained consistent across most months, with the highest sales recorded in April 2024.
